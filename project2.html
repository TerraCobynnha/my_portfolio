My Work

Return Identification Records Management

This project application provides a platform for the management of national identity records. The administrator of the identity record management accesses the stored records using an email address or username and a password. On successful login, the administrator is given the privilege to create, view, edit and remove records about the citizens. The administrator can see a count of the records available. The format of the records when viewed can be used as identification cards. Technologies used include: HTML, CSS, Bootstrap 4, jQuery, AJAX, PHP, MySQL.
